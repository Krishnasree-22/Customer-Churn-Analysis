# Customer-Churn-Analysis

##  Project Overview
This Power BI project focuses on **Customer Churn Analysis**, aiming to identify the key drivers behind customer attrition and provide actionable insights to improve customer retention strategies.

The dashboard offers an interactive view of churn metrics, customer demographics, contract types, and payment methods, helping businesses understand patterns and make data-driven decisions.

---

#  Tools & Technologies Used
- **Power BI Desktop**
- **Power Query** for data cleaning and transformation  
- **DAX (Data Analysis Expressions)** for calculated columns and measures  
- **Excel / CSV Dataset** for input data  

---

##  Project Objectives
- Analyze the overall **customer churn rate**  
- Identify key **demographic factors** affecting churn  
- Compare churn by **tenure, contract type, and payment method**  
- Highlight **churn trends** across different customer segments  
- Provide **KPIs** and **visual insights** to support retention strategies  

---

##  Key Insights
- Customers with **month-to-month contracts** show the highest churn rate.  
- **Electronic check** payment users are more likely to churn compared to other payment types.  
- Customers with **shorter tenures** have higher churn likelihood.  
- **Senior citizens** show slightly higher churn than younger customers.  
- Higher **monthly charges** correlate with increased churn tendency.

##  Dashboard Features

- KPIs: **Total Customers**, **Churned Customers**, **Churn Rate**, **Avg. Monthly Charges**, **Avg. Tenure**  
- Clear visuals showing relationships between churn and customer demographics
